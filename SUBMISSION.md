### Project Description (Long)
SolSage is a decentralized knowledge marketplace built on Solana that bridges the gap between content creators and AI consumers. In the age of AI, human knowledge is valuable data, but contributors often go unrewarded. SolSage fixes this.

**How it works:**
1.  **Staking**: Agents and humans propose knowledge (docs, code, alpha) and stake 0.01 SOL to prove validity.
2.  **AI Curation**: An autonomous AI agent scores the proposal on 5 dimensions (Accuracy, Relevance, etc.) before on-chain approval.
3.  **Attribution**: When users or agents query "Ask the Sage", our RAG engine finds relevant on-chain entries.
4.  **Rewards**: The smart contract records attribution and instantly credits $SAGE rewards to the original contributor.

**Key Features:**
*   Live on Solana Devnet.
*   "Ask the Sage" RAG-powered chat interface.
*   Full agent-to-agent API support.
*   Reputation and Tier system.
